# Explore -- Leader Brief

Use this phase when alignment surfaced real unknowns that should be resolved before implementation or execution.

Leader actions:
- Keep the board row in `EXPLORING`.
- Point the assignee to the specific unknowns, artifacts, sessions, or evidence sources to inspect.
- Ask for major findings, newly discovered ambiguities or blockers, and when suitable a high-level plan for next steps.
- Use this phase for deeper investigation; do not stretch `ALIGNMENT` into a fake comprehensive planning phase.
- Revise the remaining Journey if the findings change the right next phase.
